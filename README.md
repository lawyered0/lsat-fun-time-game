# LSAT Fun Time Game

A small React/Vite game with original LSAT-style Logical Reasoning questions.

Live app: [lawyereds-lsat-game.vercel.app](https://lawyereds-lsat-game.vercel.app)

## Run locally

```bash
npm install
npm run dev
```

Then open the local Vite URL shown in the terminal.

## Build

```bash
npm run build
```

## Notes

- The questions are original and are not official LSAT questions.
- The app is built with React and Vite.
